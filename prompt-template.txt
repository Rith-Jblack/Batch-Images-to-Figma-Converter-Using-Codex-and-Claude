1. **NO `<filter>`** — no feDropShadow, feGaussianBlur. Figma rasterizes them = blurry text.
2. **Emoji icons in main UI** — use `<text>` with Unicode, NOT `<symbol>` paths:

3. **No duplicates in main UI** — each element ONCE. No `opacity="0"` groups.
4. **Gradients** — use `<linearGradient>` for backgrounds and action buttons, not flat colors.
5. **Subtle borders** — `stroke-width="1.5" opacity="0.15"` on containers.
6. **viewBox** wide enough for both panels AND tall enough for the variants panel. The RIGHT panel is usually taller than the LEFT. Calculate: last variant y-position + ~80px. Typically `viewBox="0 0 1160 680"` or taller.
